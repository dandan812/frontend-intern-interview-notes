DOM 树层（Document Object Model）
    HTML 解析生成 DOM 树
CSSOM 树层（CSS Object Model）
    CSS 解析生成 CSSOM 树
渲染树层（Render Tree）
    DOM + CSSOM 合并，形成可见节点树
    计算每个节点的布局（Layout / Reflow）
图层（Layer / Painting Layer）
    将渲染树分割成多个图层
    每个图层独立绘制（Paint）
合成层（Compositing Layer）
    GPU 将各个图层合成显示
    支持硬件加速、动画优化 


| 分类         | CSS 属性 / 特性         | 说明                         |
| ---------- | ------------------- | --------------------------       |
| **触发合成层**  | transform           | 2D/3D 变换                   |
|            | will-change         | 提前声明即将变化的属性             |
|            | filter              | blur、brightness、contrast 等    |
|            | backface-visibility | 隐藏/显示元素背面                 |
|            | opacity ≠ 1         | 半透明元素会触发合成层             |
|            | position: fixed     | 固定定位元素通常单独合成           |
| **不触发合成层** | overflow            | 隐藏/滚动不触发 GPU          |
|            | display             | 修改布局，不触发 GPU               |
|            | visibility          | 隐藏/显示元素，不触发 GPU          |
|            | z-index             | 层叠顺序变化，不触发 GPU           |

修改几何属性 → 重排 → 最卡  
修改视觉属性 → 重绘 → 中等  
只改合成属性 → 合成 → 最流畅
