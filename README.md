# Frontend Intern Interview Notes

前端实习面试复盘与知识沉淀仓库（持续更新）

---

## 一、仓库说明

本仓库用于系统性记录 **前端实习面试过程中的问题、复盘与改进方案**。

在准备前端实习的过程中，我会对每一次面试进行完整复盘，包括：
- 被问到的技术问题
- 当时的真实回答
- 面试官的追问点
- 面试后整理的更优回答和改进计划

目标是做到：**每面一次，能力迭代一次**。

---

## 二、仓库用途

本仓库主要用于：

- 📌 前端实习面试复盘与总结  
- 📌 高频前端基础知识整理  
- 📌 手写题与常见考点沉淀  
- 📌 项目相关问题与表达能力提升  
- 📌 面试前快速回顾与查漏补缺
