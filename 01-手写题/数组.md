1️⃣ 数组去重

使用 Set
const unique = [...new Set(arr)];

2️⃣ 数组扁平化
flat（ES6）
arr.flat(Infinity);

reduce + concat
arr.reduce((a,b)=>a.concat(Array.isArray(b)?b.flat():b),[]);

3️⃣ 数组求和

reduce
const sum = arr.reduce((total, n) => total + n, 0);
4️⃣ 数组查找元素

find 找到第一个符合条件的元素
arr.find(x => x > 10);


findIndex 找到元素索引
arr.findIndex(x => x > 10);


some 判断至少有一个元素满足条件
arr.some(x => x > 10);


every 判断所有元素是否满足条件
arr.every(x => x > 0);

5️⃣ 数组合并与扁平化操作

concat 合并数组
[1,2].concat([3,4]); // [1,2,3,4]


flatMap 扁平化 + map
[1,2].flatMap(x => [x, x*2]); // [1,2,2,4]

6️⃣ 数组排序

升序/降序
arr.sort((a,b)=>a-b); // 升序
arr.sort((a,b)=>b-a); // 降序

7️⃣ 数组转字符串 / 字符串转数组

join
[1,2,3].join('-'); // '1-2-3'


Array.from / split
Array.from('abc'); // ['a','b','c']
'1,2,3'.split(','); // ['1','2','3']


const arr = [1, 2, 3, 4];
const str = arr.join(','); 
console.log(str); // "1,2,3,4"


const arr = [
  { name: 'Alice', age: 20 },
  { name: 'Bob', age: 22 },
  { name: 'Charlie', age: 25 }
];

// 取 name 属性，用逗号拼接
const str = arr.map(item => item.name).join(',');
console.log(str); // "Alice,Bob,Charlie"

3️⃣ ES6 解构写法（更简洁）
const str = arr.map(({name}) => name).join(',');
console.log(str); // "Alice,Bob,Charlie"
